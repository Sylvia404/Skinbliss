#  Skinbliss Tanzania

> *Skin that feels like bliss* – Handcrafted skincare made in Tanzania.

This is the complete source code for the Skinbliss store, featuring:
-  Dynamic product and package management
-  Image uploads for products, founder photo, and Instagram tiles
-  Cart & checkout with M‑Pesa, Tigo Pesa, Airtel Money, and Cash on Delivery
-  Hidden admin panel (double‑click the brand name)
-  Dark / light mode

## Tech stack
- PHP + MySQL
- Vanilla HTML / CSS / JavaScript
- Git for version control

## Setup
1. Import the SQL schema (see code comments)
2. Create `uploads/` folder with write permissions
3. Update database credentials in `index.php`
4. Deploy to your server and enjoy!
